Recording Demo Video — Instructions

1. Prepare the environment: run backend and open `FRONTEND/admin/admin.html` in a browser.
2. Use screen recorder (OBS, QuickTime, or built-in) set to 1080p.
3. Follow the demo script in `docs/demo_video_script.md` and use `docs/demo_voiceover.md` for narration.
4. Keep clips short; show API calls in the terminal if needed.
5. Edit and export final clip as MP4, keep under 2 minutes.

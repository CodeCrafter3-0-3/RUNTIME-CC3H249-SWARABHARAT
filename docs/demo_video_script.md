Demo Video Script (90-120s)

1. (0-10s) Title slide: "SwaraBharat — Civic Pulse powered by AI" with one-line problem statement.
2. (10-25s) Show the admin dashboard landing: total voices, top issue, and emotion chart.
3. (25-50s) Show `GET /demo_examples` in the UI: load 2 sample citizen messages and show the AI's structured analysis (issue, emotion, urgency, summary).
4. (50-70s) Submit a live message via the analyze form and show the returned analysis; explain anonymization and safety.
5. (70-90s) Show how data aggregates in the dashboard and mention deployment / scalability plan.
6. (90-120s) Closing: impact metrics you aim to measure (response time, accuracy, % high-urgency detected), call to action.

Tips:
- Narration should emphasize privacy: only anonymized insights are stored.
- Use captions and highlight the AI output visually.
- Keep transitions smooth and keep each step short and focused.
